## Pixel Art Drawer

Pixel Art Drawer that lets you draw pixel art with canvas size ranging from 1x1 to 50x50.\
You can save your drawing locally by downloading it as a .png 

## [Live Demo](https://ngoytom.github.io/pixel-art-drawer/)
